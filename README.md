Electronics Shop E-Commerce Platform
This repository contains a full-stack E-Commerce Web Application developed for the EWA408510 - E-Commerce and Web Application final examination (Academic Year 2025-2026) at UNILAK.

🚀 Project Overview
This platform provides a modern, responsive interface for users to browse high-end electronic products, manage a shopping cart, and perform secure checkouts. The project emphasizes professional design, robust database integration, and modern DevOps practices.

🛠 Tech Stack
Frontend: HTML5, EJS (Embedded JavaScript Templates), Tailwind CSS

Backend: Node.js, Express.js

Database: MongoDB

Containerization: Docker & Docker Compose

DevOps/CI/CD: GitHub Actions

📋 Core Features
Responsive UI: Mobile-friendly design optimized for all devices.

Product Management: Dynamic catalog with category filtering and detailed technical specifications.

Shopping Cart: Real-time quantity updates and price calculations using LocalStorage.

Checkout Process: Streamlined order capture and data validation.

Database Integration: Document-based storage for Products, Customers, and Orders.

🐳 Deployment & Containerization
This project is fully containerized to ensure environment consistency.
